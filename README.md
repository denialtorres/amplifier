## Webpacker
Node version: v15.14.0
```
nvm use 15.14.0
```
1- Add webpacker gem: `gem 'webpacker'`
2- Install webpacker from terminal: `rake webpacker:install`
3- Run command `yarn build`

run in diferent terminals
```
bin/webpack-dev-server
```
